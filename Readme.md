# My Wine Collection 

## Version 1.0

## Description
> Small wine collection web app that enables you to create and delete wines.

## Tech used
[ReactJS](https://reactjs.org/)
[Jest](https://facebook.github.io/jest/)
[Enzyme](https://github.com/airbnb/enzyme)

## Install notes

> Navigate to project folder
> Run npm install

## Run project
npm run dev

## Run Test
npm run test
